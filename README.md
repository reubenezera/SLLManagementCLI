# SLL Management CLI
A simple interactive command-line interface (CLI) that allows node insertion and deletion in a singly linked list (SLL).

## How to run:
### 1. Copy the code in ```main.c```.
### 2. Open [Programmiz's Online C Compiler](https://www.programiz.com/c-programming/online-compiler/).
### 3. Paste the code in the compiler.
### 4. Run the code.
